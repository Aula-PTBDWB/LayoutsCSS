# Portal IFSP - Câmpus Pirituba 🏫

Este projeto é uma aplicação web estática desenvolvida para o **Instituto Federal de Educação, Ciência e Tecnologia de São Paulo (IFSP)**, especificamente para o câmpus Pirituba. O site serve como um portal de informações sobre cursos, eventos institucionais e canais de atendimento.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias fundamentais do desenvolvimento web:

* **HTML5**: Estruturação de conteúdo, formulários de inscrição e semântica.
* **CSS3**: Estilização visual, sistema de grid para o layout e componentes de menu dropdown.
* **Vídeos e Imagens Externas**: Integração com YouTube e recursos visuais hospedados no GitHub.

## 📋 Funcionalidades

* **Menu de Navegação Responsivo**: Inclui um menu superior com efeito *hover* e um sistema de *dropdown* para a listagem de cursos.
* **Página Inicial (Home)**: Exibe banners informativos, notícias recentes organizadas em grid e um vídeo institucional.
* **Gestão de Cursos**: Páginas individuais detalhando os objetivos de cursos como:
    * Análise e Desenvolvimento de Sistemas (ADS).
    * Engenharia de Produção.
    * Gestão em TI.
    * Letras.
    * Gestão Pública.
* **Sistemas de Formulários**:
    * **Contato**: Formulário com validação de campos (E-mail, Telefone).
    * **Inscrição em Eventos**: Formulário completo com seleção de eventos, dados pessoais e campos de perfil (URL do GitHub, cor favorita, etc.).
* **Área de Acesso**: Telas de Login e Recuperação de Senha.
